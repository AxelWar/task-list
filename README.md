# To-Do List Check

This project was created as an interview challenge for https://clarikagroup.com/


## Tech Stack

**Client:** Angular 17

**CI/CD:** Netlify

## Features

- SOLID Principles.
- Libraries: ESLint, Prettier.
- Package Manager: NPM.
- Testing: JEST


## Run Locally

Clone the project

```bash
  git clone https://github.com/AxelWar/task-list.git
```

Go to the project directory

```bash
  cd task-list
```

Install dependencies

```bash
  npm install
```

Start the client

```bash
  npm run start
```
## Running Tests

To run tests, run the following command

```bash
  npm run test
```
